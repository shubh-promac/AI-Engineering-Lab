# 🚀 AI Engineering Lab
This repository is a hands-on guide for learning AI, Data Science, and Machine Learning from basics to advanced topics
To get started, clone the repo and explore the folders by topic; contributions and suggestions are welcome via pull requests or issues.

## 🗺️ Learning Roadmap
- [x] **Phase 1: Data Foundations** (NumPy, Statistics, Math)
- [ ] **Phase 2: Data Manipulation** (Pandas, Data Cleaning)
- [ ] **Phase 3: Data Visualization** (Matplotlib, Seaborn)
- [ ] **Phase 4: Machine Learning** (Scikit-Learn)
- [ ] **Phase 5: Deep Learning & AI** (Neural Networks, TensorFlow/PyTorch)

## 📂 Repository Contents

### 🔢 01. NumPy & Math
*Notes and scripts covering array manipulation and statistical analysis.*

*Mastering the art of data storytelling and graph making.*

*Future home for dataframes and real-world CSV analysis.*
